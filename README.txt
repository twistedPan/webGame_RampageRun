Titel: Rampage Run
Developer: Dennis Düblin
Modul: DEWEBMOB / HS / 2020
Beschreibung:
Zu Lernzwecken erhielten die Studierenden die Aufgabe ein Web-Game zu erstellen. Dabei sollte, wenn möglich auf externe Bibliotheken verzichtet werden. In dieser Arbeit befindet man sich in der Rolle eines Randale, der sich auf der Flucht vor dem Gesetz befindet. Dabei müssen umherstreifende Helikopter mittels anderen Autos abgeschossen werden.
Ziel ist es vor Ablauf der Zeit so viele Autos und Helikopter wie möglich zu treffen.
